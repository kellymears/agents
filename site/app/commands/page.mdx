## /cpr - Commit, Push, Pull Request

A complete git workflow command that handles committing, pushing, and creating a pull request in one step.

### What It Does

1. **Commits** your changes with a conventional commit message
2. **Pushes** to the remote repository
3. **Creates** a GitHub Pull Request via `gh pr create`

### Commit Format

Uses conventional commits format:

```
type(scope): description
```

### Example

```bash
# After making changes to your code
/cpr
```

Claude will then:

1. Analyze your changes
2. Generate an appropriate commit message
3. Push to your current branch
4. Create a PR with a descriptive title and body

---

## Creating Your Own Commands

Want to add a custom command? Create a markdown file in `.claude/commands/`:

```bash
.claude/commands/my-command.md
```

The command will be available as `/my-command` in Claude Code.
