## Install

Add the marketplace:

```
/plugin marketplace add kellymears/agents
```

Then install individual plugins by name:

```
/plugin install git@kellymears              # commit curation, PRs, issues, context switching
/plugin install research@kellymears         # deep-dive multi-source research
/plugin install comms@kellymears            # draft emails, Slack messages, blog posts
/plugin install playwright-cli@kellymears   # browser automation
/plugin install progressive-sim@kellymears  # timed coding assessments
/plugin install sprint-worker@kellymears    # batch-process GitHub issues
/plugin install tdd@kellymears              # red-green-refactor TDD cycle
/plugin install obsidian@kellymears         # write notes to Obsidian vaults
```
