## Custom Commands

### /cpr — Commit, Push, Pull Request

A streamlined git workflow command that handles the entire flow from commit to PR creation.

**Features:**
- Conventional commit format (`type(scope): description`)
- Automatic push to remote
- Creates GitHub PR via `gh pr create`

**Commit types:** `feat` | `fix` | `docs` | `style` | `refactor` | `test` | `chore`

---

## Custom Agents

### The Archivist

A meticulous curator of git history that transforms messy changes into clean, well-scoped commits.

**Capabilities:**
- Organizes uncommitted changes into logical commits
- Consolidates and reorders recent commits
- Uses conventional commit format with meaningful messages
